# COMET
Comet is an RS-232 Serial Port Program that builds a UI for a device in real time. Each serial command entered creates a History Button so quick-access, reuse, and automation. 
Users can save and load past sessions, and modify simple text files to create UIs for RS-232, RS 485, test equipment, embedded devices, 4G and 5G modems, Arduino - any device that accepts serial input.
